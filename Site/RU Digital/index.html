Redes: atualidade

O uso de papel para tal controle é antiquado. A necessidade de anotar o nome para o próximo dia força que os alunos compareçam todos os dias para anotar o nome. Com a computação tudo isso pode ser simplificado com um app: R.U. Digital!

A solução

Para resolver tal problema dois estudantes da cadeira de Redes decidiram criar um aplicativo único, integrado ao banco de dados e de fácil acesso: o R.U. Digital. Um app inovador e de relevância, feito por estudantes para estudantes!
